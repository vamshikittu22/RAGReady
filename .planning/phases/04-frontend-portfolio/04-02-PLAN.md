  <action>
    Build the chat interface — the core feature of the app. Requirements: UI-01, UI-02, UI-03.

    1. **ConfidenceBadge** (`shared/confidence-badge.tsx`):
       - Accepts `confidence: number` prop (0.0-1.0)
       - Uses shadcn Badge component with color variants:
         - 0.8-1.0 → green (destructive=false, custom green classes) label "High Confidence"
         - 0.5-0.79 → amber/yellow label "Medium Confidence"
         - 0.0-0.49 → red label "Low Confidence"
       - Display percentage: `Math.round(confidence * 100)%`

    2. **ChatInput** (`chat/chat-input.tsx`):
       - Textarea (shadcn) + Send button (shadcn Button with Lucide Send icon)
       - Submit on Enter (not Shift+Enter which adds newline)
       - Disable input + button while query is pending (accept `isPending` prop)
       - Call `onSubmit(question: string)` prop when user sends
       - Clear input after submit
       - Placeholder: "Ask a question about your documents..."

    3. **MessageBubble** (`chat/message-bubble.tsx`):
       - Accepts `message: ChatMessage` prop (ChatMessage type from types/api.ts created in Plan 01)
       - User messages: right-aligned, primary background
       - Assistant messages: left-aligned, muted background
       - If assistant message: show ConfidenceBadge below the answer text
       - If `refused === true`: render with amber/warning Alert component showing refusal reason, no confidence badge needed
       - Clickable: clicking an assistant message calls `onSelect(message)` to show its citations in the sidebar

    4. **MessageList** (`chat/message-list.tsx`):
       - Accepts `messages: ChatMessage[]` and `selectedMessageIndex: number | null`
       - Uses shadcn ScrollArea for scrollable container
       - Renders MessageBubble for each message
       - Auto-scrolls to bottom when new message is added (useEffect + ref with `scrollIntoView({ behavior: 'smooth' })`)
       - Shows a loading skeleton (Skeleton component) when `isLoading` prop is true (simulates assistant "typing")
       - Empty state: centered text "Ask a question to get started"

    5. **CitationsPanel** (`chat/citations-panel.tsx`):
       - Accepts `citations: Citation[]` prop (from selected message)
       - Renders in sidebar area (the parent controls layout)
       - For each citation:
         - Document name as a bold label/badge
         - Page number (or "N/A" if null)
         - Relevance score as percentage badge (e.g., "92%")
         - Chunk text: show first 200 characters, expandable on click to show full text
       - Sort citations by relevance_score descending
       - Empty state when no message selected: "Select a message to view citations"
       - Use shadcn Card for each citation entry, Separator between them

    6. **ChatPanel** (`chat/chat-panel.tsx`):
       - This is the main orchestrator component for the chat tab
       - Manages chat state: `messages: ChatMessage[]` in useState (in-memory, no persistence)
       - Manages selected message index for citations sidebar
       - Uses `useQueryRag()` hook (from Plan 01's hooks)
       - Layout: flex row — main content (70%, MessageList + ChatInput at bottom) | sidebar (30%, CitationsPanel)
       - On submit: add user message to state → call `ask(question)` → on success, add assistant message with response data (answer, citations, confidence, refused)
       - On refusal: add assistant message with `refused: true` and `refusalReason`
       - Error handling: if mutation fails, show toast notification via sonner
       - Responsive: on small screens, stack vertically (citations below chat)
  </action>
