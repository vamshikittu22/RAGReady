  <action>
    Implement 6 metric modules in `tests/evaluation/metrics/`. Every metric function takes evaluation results and golden dataset entries, returns a float score between 0.0 and 1.0.

    **1. faithfulness.py — `compute_faithfulness(responses, contexts)`**

    Try to use Ragas `FaithfulnesswithHHEM` first. The import path may be:
    - `from ragas.metrics import FaithfulnesswithHHEM`
    - Or `from ragas.metrics._faithfulness import FaithfulnesswithHHEM`

    If HHEM is not available in the installed Ragas version, implement a custom HHEM wrapper:
    - Use `transformers` to load `vectara/hallucination_evaluation_model` (HHEM-2.1-Open)
    - For each (answer, context) pair, classify as hallucination or faithful
    - Score = mean of faithful classifications

    Fallback (if neither Ragas HHEM nor transformers HHEM works): use a simple NLI approach with sentence-transformers — compute cosine similarity between each answer sentence and the concatenated context. Sentences with similarity > 0.5 are "faithful." This is less accurate but deterministic and always available.

    Signature: `def compute_faithfulness(responses: list[QueryResponse], contexts: list[list[ScoredChunk]]) -> float`

    Also derive hallucination rate: `def compute_hallucination_rate(faithfulness_score: float) -> float` — simply `1.0 - faithfulness_score`.

    **2. relevancy.py — `compute_answer_relevancy(questions, responses, model)`**

    Embedding-based answer relevancy:
    1. Load `all-MiniLM-L6-v2` model (from sentence-transformers, already a dependency)
    2. For each (question, answer) pair: embed both, compute cosine similarity
    3. Return mean cosine similarity across all pairs
    4. Skip RefusalResponse entries (they don't have answers to score)

    Cosine similarity: `numpy.dot(a, b) / (numpy.linalg.norm(a) * numpy.linalg.norm(b))`

    Signature: `def compute_answer_relevancy(questions: list[str], responses: list[QueryResponse], model: SentenceTransformer) -> float`

    **3. context_recall.py — `compute_context_recall(golden_entries, retrieved_contexts, model)`**

    Embedding-based context recall:
    1. For each golden entry with `expected_contexts`: embed all expected context snippets
    2. Embed all actually retrieved chunk texts
    3. An expected context is "recalled" if any retrieved chunk has cosine similarity > 0.7 with it
    4. Context Recall = (recalled expected contexts) / (total expected contexts)
    5. Skip entries with `should_refuse=True` (no expected contexts to check)

    Signature: `def compute_context_recall(golden_entries: list[dict], retrieved_contexts: list[list[ScoredChunk]], model: SentenceTransformer) -> float`

    **4. context_precision.py — `compute_context_precision(questions, retrieved_contexts, model)`**

    Embedding-based context precision:
    1. For each question: embed the question
    2. Embed each retrieved chunk
    3. A chunk is "relevant" if cosine similarity with question > 0.3 (lower threshold — measures general relevance)
    4. Context Precision = (relevant chunks) / (total retrieved chunks)
    5. Skip entries where no chunks were retrieved

    Signature: `def compute_context_precision(questions: list[str], retrieved_contexts: list[list[ScoredChunk]], model: SentenceTransformer) -> float`

    **5. refusal_accuracy.py — `compute_refusal_accuracy(golden_entries, responses)`**

    Fully deterministic:
    1. For each golden entry:
       - If `should_refuse=True`: correct if response is RefusalResponse (or has `refused=True`)
       - If `should_refuse=False`: correct if response is QueryResponse (not RefusalResponse)
    2. Refusal Accuracy = correct / total

    Handle both QueryResponse and RefusalResponse types using isinstance checks.

    Signature: `def compute_refusal_accuracy(golden_entries: list[dict], responses: list[QueryResponse | RefusalResponse]) -> float`

    **6. citation_accuracy.py — `compute_citation_accuracy(responses, retrieved_contexts, known_documents)`**

    Fully deterministic citation validation:
    1. For each QueryResponse (skip RefusalResponse):
       a. Every citation's `document_name` must exist in `known_documents` set
       b. Every citation's `chunk_text` must appear as substring in at least one retrieved chunk's text
    2. Citation Accuracy = valid_citations / total_citations
    3. Return 1.0 if no citations exist (edge case)

    Signature: `def compute_citation_accuracy(responses: list[QueryResponse | RefusalResponse], retrieved_contexts: list[list[ScoredChunk]], known_documents: set[str]) -> float`

    **IMPORTANT for all metrics:**
    - Each module should have a module-level docstring explaining what it measures, how it works, and what its limitations are
    - All functions should handle empty inputs gracefully (return 1.0 or 0.0 as appropriate)
    - Use type hints throughout
    - No LLM calls in any metric — all are deterministic or classifier-based
