    Signature: `def compute_answer_relevancy(questions: list[str], responses: list[QueryResponse | RefusalResponse], model: SentenceTransformer) -> float`

    **3. context_recall.py — `compute_context_recall(golden_entries, retrieved_contexts, model)`**

    Embedding-based context recall:
    1. Accept pre-loaded SentenceTransformer model
    2. For each golden entry with `expected_contexts`: embed all expected context snippets
    3. Embed all actually retrieved chunk texts
    4. An expected context is "recalled" if any retrieved chunk has cosine similarity > 0.7 with it
    5. Context Recall = (recalled expected contexts) / (total expected contexts)
    6. Skip entries with `should_refuse=True` (no expected contexts to check)

    Signature: `def compute_context_recall(golden_entries: list[dict], retrieved_contexts: list[list[ScoredChunk]], model: SentenceTransformer) -> float`

    **4. context_precision.py — `compute_context_precision(questions, retrieved_contexts, model)`**

    Embedding-based context precision:
    1. Accept pre-loaded SentenceTransformer model
    2. For each question: embed the question
    3. Embed each retrieved chunk
    4. A chunk is "relevant" if cosine similarity with question > 0.3 (lower threshold — measures general relevance)
    5. Context Precision = (relevant chunks) / (total retrieved chunks)
    6. Skip entries where no chunks were retrieved

    Signature: `def compute_context_precision(questions: list[str], retrieved_contexts: list[list[ScoredChunk]], model: SentenceTransformer) -> float`

    **5. refusal_accuracy.py — `compute_refusal_accuracy(golden_entries, responses)`**

    Fully deterministic:
    1. For each golden entry:
       - If `should_refuse=True`: correct if response is RefusalResponse (or has `refused=True`)
       - If `should_refuse=False`: correct if response is QueryResponse (not RefusalResponse)
    2. Refusal Accuracy = correct / total

    Handle both QueryResponse and RefusalResponse types using isinstance checks or checking for `refused` attribute.

    Signature: `def compute_refusal_accuracy(golden_entries: list[dict], responses: list[QueryResponse | RefusalResponse]) -> float`

    **6. citation_accuracy.py — `compute_citation_accuracy(responses, retrieved_contexts, known_documents)`**

    Fully deterministic citation validation:
    1. For each QueryResponse (skip RefusalResponse):
       a. Every citation's `document_name` must exist in `known_documents` set
       b. Every citation's `chunk_text` must appear as substring in at least one retrieved chunk's text (or vice versa — account for truncation)
    2. Citation Accuracy = valid_citations / total_citations
    3. Return 1.0 if no citations exist (edge case — nothing to be wrong about)

    Signature: `def compute_citation_accuracy(responses: list[QueryResponse | RefusalResponse], retrieved_contexts: list[list[ScoredChunk]], known_documents: set[str]) -> float`

    **IMPORTANT for all 4 modules:**
    - Module-level docstring explaining what it measures
    - Handle empty inputs gracefully (return 1.0 or 0.0 as appropriate)
    - Use type hints throughout
    - No LLM calls
